# うさうさふぁーむ

小さなうさぎをなでて増やしていく、スマホ縦持ち向けの1画面クリッカーゲームです。

## 遊び方

`index.html` をブラウザで開くと起動します。中央のうさぎを「なでる」と、うさぎの数が増えます。増えたうさぎを消費して、にんじん強化またはなでる手強化を進めます。

- にんじん強化: うさぎの毎秒増加量を上げます。
- なでる手強化: なでた時の増加量を上げます。
- SEサンプル: タップ音のサンプルを再生します。
- 音色 A/B/C: 後半で音程が上がるSEサンプルを切り替えます。
- リセット: 保存済みの進行状況を初期化します。

## にんじん段階

ふつう、ルビー、ガーネット、トパーズ、エメラルド、サファイア、ダイヤの順に解放されます。

## 開発

ビルド手順はありません。静的ファイルのみで動作します。

```text
index.html
styles.css
app.js
```

動作確認は `index.html` を直接ブラウザで開いて行います。
